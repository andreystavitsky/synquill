# Synquill

A Flutter package for Drift-powered offline storage with automatic REST and GraphQL API synchronization.

## Features

- **Offline-first** data persistence with Drift.
- **Automatic two-way synchronization** with REST and GraphQL APIs.
- **JsonSerializable** model support.
- **Pluggable API adapters** (REST and GraphQL supported out-of-the-box).
- **Background sync** and retry mechanisms.
